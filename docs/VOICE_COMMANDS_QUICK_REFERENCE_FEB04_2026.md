## Tips for Best Results

1. **Wait for connection**: First connection takes 60-90 seconds for CUDA warmup
2. **Speak clearly**: Normal pace, clear pronunciation
3. **Use keywords**: Commands work better with key phrases
4. **Check status**: Look for green indicator before speaking
5. **Be specific**: "Show agent orchestrator" works better than "show it"

## Troubleshooting

**Voice not working?**
1. Check mic permissions in browser
2. Ensure PersonaPlex Bridge is running (port 8999)
3. Wait for CUDA warmup (60-90 seconds)
4. Try refreshing the page

**Commands not recognized?**
1. Speak more clearly
2. Use exact phrases from this guide
3. Check for background noise
4. Try shorter commands
