## Core Actions
### Action 1: Review Situational Awareness Picture
**Goal:** Understand the current fused environmental and security picture
1. Navigate to /natureos/fusarium
2. Review the unified map view showing all fused data layers
3. Check the AI inference panel for automated assessments
4. Note any elevated threat indicators or biosecurity alerts
5. Cross-reference with raw CREP data if needed

### Action 2: Monitor Data Fusion Status
**Goal:** Verify all data streams are feeding the fusion engine
1. Check the Data Stream Status Panel
2. Verify CREP streams are active (fungal, environmental, transport)
3. Verify edge sensor data (MycoBrain devices) is flowing
4. Check OEI connector feeds are integrated
5. Note any streams showing degraded or offline status

### Action 3: Review AI Inference Results
**Goal:** Examine AI-generated situational assessments
1. Open the AI Inference Panel
2. Review current threat assessments and confidence levels
3. Check for anomaly detection alerts
4. Review biosecurity risk scoring
5. Examine species identification results from sensor fusion

### Action 4: Check Neuromorphic Computing Integration
**Goal:** Verify edge computing inference pipeline
1. Locate the Neuromorphic Computing Status panel
2. Check edge device inference latency and throughput
3. Verify model deployment status across edge nodes
4. Review inference accuracy metrics

## Common Failure Modes
| What You See | What Went Wrong | What To Do |
|---|---|---|
| "Data Fusion Offline" | Backend fusion engine crashed or disconnected | Check service health; restart fusion pipeline |
| Map shows partial data | One or more data streams disconnected | Check Data Stream Status; use OEI monitoring to diagnose |
| AI inference "Stale" | Inference pipeline backlogged or crashed | Check AI service logs; verify model is loaded |
| Neuromorphic status "Disconnected" | Edge devices offline or network issue | Check device fleet for connectivity issues |
| Access denied | Not COMPANY-tier authenticated | Re-login with COMPANY-level credentials |
