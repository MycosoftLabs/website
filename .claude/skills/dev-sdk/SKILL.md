# NatureOS Developer SDK

## Identity
- **Category**: dev
- **Access Tier**: AUTHENTICATED access
- **Depends On**: platform-authentication, platform-navigation
- **Route**: /natureos/sdk
- **Key Components**: SDK documentation pages, code examples, integration guides, SDK download/install instructions

## Success Criteria (Eval)
- [ ] SDK page loads at /natureos/sdk
- [ ] SDK documentation is browsable and readable
- [ ] Code examples are displayed with syntax highlighting
- [ ] Integration guides provide step-by-step instructions
- [ ] SDK download or install command is accessible

## Navigation Path (Computer Use)
1. Log in with any authenticated account
2. Navigate to NatureOS section
3. Click "SDK" or go directly to /natureos/sdk
4. Browse documentation sections

## Screen Elements Map
| What You'll See | Where On Screen | What To Do |
|---|---|---|
| SDK heading / NatureOS Developer SDK | Top of page | Confirms correct page |
| Documentation sidebar | Left side — table of contents | Navigate between SDK sections |
| Getting Started section | First content section | Begin here for initial setup |
| Code examples | Inline code blocks with syntax highlighting | Copy and adapt for your project |
| Integration guides | Dedicated guide pages | Follow step-by-step instructions |
| SDK download button/command | Prominent placement — top or Getting Started | Download or copy install command |
| API reference link | In sidebar or navigation | Links to dev-api-gateway |
| Language/framework tabs | Above code examples | Switch between language variants |

## Core Actions
### Action 1: Get Started with the SDK
**Goal:** Install and configure the NatureOS SDK
1. Navigate to /natureos/sdk
2. Find the Getting Started section
3. Copy the SDK install command or download link
4. Follow the configuration steps (API key setup, endpoint configuration)
5. Run the verification/test command to confirm setup

### Action 2: Browse Code Examples
**Goal:** Find code snippets for specific SDK functionality
1. Navigate through the documentation sidebar
2. Select a topic (e.g., device management, data queries, species lookup)
3. Review the code example with syntax highlighting
4. Switch between language/framework tabs if available
5. Copy the code snippet for your project

### Action 3: Follow Integration Guide
**Goal:** Integrate the SDK into an existing application
1. Select the relevant integration guide from the sidebar
2. Read the prerequisites and requirements
3. Follow the step-by-step instructions
4. Reference code examples at each step
5. Test the integration using provided verification steps

## Common Failure Modes
| What You See | What Went Wrong | What To Do |
|---|---|---|
| Access denied / login redirect | Not authenticated | Log in with any valid account |
| Documentation page blank | Content failed to load or build | Refresh the page; clear browser cache |
| Code examples show errors | SDK version mismatch or deprecated API | Check SDK version; look for updated examples |
| Download link broken | SDK artifact not available | Check releases page; contact support |
| "API key required" in examples | Need to obtain an API key first | Go to dev-api-gateway to create an API key |

## Computer Use Notes
- SDK documentation uses a sidebar navigation pattern — scroll the sidebar to find topics
- Code examples may have a "Copy" button in the top-right corner of code blocks
- Language tabs (Python, JavaScript, etc.) appear above code blocks — click to switch
- The install command is typically a pip/npm/cargo command displayed prominently
- Documentation pages may be long — scroll down for complete content
- Cross-references to the API Gateway are common — those link to /natureos/api
